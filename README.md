# CamCalc
CamCalc is an open-source Python utility for stop-motion animators.

### please note-
CamCalc currently runs from the command line and requires Python to be installed.


## Features
- Constant camera motion calculator
- Frame calculator
- FPS calculator

  ## Planned Features
  - Motion easing
  - Manual rig calibration
  - Camera path planner
  - Lightweight 3D visualization and Planning
 
  ## Status
  Early development.
